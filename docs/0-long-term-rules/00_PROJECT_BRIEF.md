# Amber Compass Project Brief

## 产品定位
Amber Compass 是一个现代、简洁、专业的中国传统历法与命理工具站。

## 核心目标
把公历、农历、黄历与四柱八字等传统时间信息，整理成一个清晰、可信、可持续扩展的 Web 产品。

## 目标用户
对传统历法、黄历、八字排盘、时间命理决策有需求或感兴趣，并希望获得更现代、更清爽使用体验的个人用户。

## 核心体验
用户打开产品后，可以在一个风格统一、克制的界面中查看日历、黄历和八字等信息，一切内容清晰明了井井有条，而不是进入一个杂乱、玄学感过重的网站。

## 当前产品阶段
Phase 1 日历 / 黄历已经完成基础稳定化。
Phase 2 八字模块正在验证和打磨中。

## 设计风格
现代、极简、专业、克制。
参考 OpenAI / Claude / Google 的清爽产品感，同时保留温润的人文气质。

## 不做什么
- 不做夸张玄学包装
- 不做复杂动效
- 不做无边界功能堆叠
- 不把技术解释堆在主界面
- 不破坏现有固定布局基线

## 当前优先级
1. 稳定现有布局基线
2. 完善八字模块的准确性与可信度
3. 统一视觉语言
4. 建立可持续的 AI 辅助开发工作流
