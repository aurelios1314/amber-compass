# Amber Compass Visual Token Standard

## 目标

建立 Amber Compass 的统一视觉 Token 系统。

确保：

- Logo
- Calendar
- Almanac
- Bazi
- Settings
- Navigation
- Empty States

都使用同一套视觉语言。

避免：

- 模块各自发展
- 局部硬编码
- UI 风格漂移
- 圆角 / 阴影 / 边框失控
- 东方感失控
- 后台系统感

---

# 一、视觉方向

Amber Compass 的视觉方向：

现代
克制
温润
可信
专业
东方秩序感

不是：

- 传统玄学网站
- 赛博东方
- 拟物风
- 科技光效风
- 后台管理系统

---

# 二、核心原则

## 1. Logo 是视觉母体

Logo 中的：

- 双圆环
- 点阵
- 琥珀色
- 山水层次
- 东方秩序感

必须转译进入 UI。

但：

只能抽象转译，
不能直接堆符号。

禁止：

- 大面积纹样
- 八卦背景
- 云纹贴图
- 发光罗盘
- 东方图腾墙

---

## 2. UI 是工作台，不是展示页

Amber Compass 是：

历法与命理工作台。

不是：

营销官网
算命娱乐页
传统黄历网站

因此：

优先级：

结构感 > 装饰感
可信感 > 氛围感
秩序感 > 神秘感

---

## 3. 视觉系统优先于局部好看

禁止：

模块单独发明风格。

所有 UI：

必须优先服从全局 Token。

---

# 三、色彩 Token

## 主背景

--bg

用途：

主工作区背景。

方向：

暖象牙白 / 亚麻纸感。

---

## 主表面

--surface

用途：

主卡片
主面板
顶部栏
主工作区

必须：

稳定
克制
低纹理感。

---

## 次级表面

--surface-dim

用途：

侧栏
次级面板
弱区域。

---

## 暖纸层

--sand
--sand-dark

用途：

日期格
黄历块
输入区
辅助信息区
hover

这是：

Amber Compass 的核心气质层。

---

## 主强调色

--cin
--cin-bg
--cin-fill

用途：

selected
active
focus
主强调状态

要求：

只作为少量重点强调。

禁止：

大面积琥珀色铺满。

---

## 文本色

--ink
--ink-2
--ink-3
--ink-4

必须严格控制层级。

禁止：

所有文本权重接近。

---

# 四、圆角 Token

允许：

--r-sm
--r
--r-md
--r-lg

建议：

4px
6px
8px
10px

禁止：

模块自由新增：

3px
5px
12px
14px
18px

特别说明：

Amber Compass 不适合超大圆角。

它应该：

精密
克制
稳定

而不是：

社交 App 感。

---

# 五、阴影 Token

阴影必须极轻。

允许：

--shadow-sm
--shadow
--shadow-lg

方向：

低对比
低透明度
短距离阴影

禁止：

- 重投影
- 大面积发光
- 彩色阴影
- 浮空卡片感

Amber Compass 应更像：

编辑级工作台。

---

# 六、边框 Token

边框是核心结构语言。

允许：

--line-light
--line
--line-dark

要求：

细
暖
低对比

禁止：

硬黑线
强分割
高科技描边

---

# 七、字体 Token

## sans

默认 UI 字体。

用于：

正文
表单
设置
导航
说明

---

## serif

只允许用于：

品牌
日期
核心干支
少量标题

禁止：

大面积 serif UI。

否则：

会快速偏传统命理网站。

---

# 八、交互状态

允许：

hover
selected
focus
disabled

必须：

轻
稳
低干扰

禁止：

- 强动画
- 弹跳
- 发光
- 粒子
- 扫描线
- 大渐变

---

# 九、空状态语言

空状态必须：

安静
可信
克制
专业

允许：

- 点阵
- 轻圆环
- 极细线条
- 淡琥珀定位点

禁止：

- 插画
- 吉祥物
- 大图标
- 玄学符号墙
- 营销文案

---

# 十、模块统一原则

Calendar
Almanac
Bazi
Settings

必须：

共享：

- 色彩
- 圆角
- 边框
- 文字层级
- 空状态语言
- hover / selected 规则

禁止：

模块各自发展。

---

# 十一、当前治理重点

当前优先级：

1. 圆角统一
2. 阴影统一
3. serif 使用收敛
4. hover / selected 统一
5. Empty State 系统建立
6. 顶栏品牌系统统一
7. 日历与八字视觉统一

---

# 十二、当前禁止事项

当前阶段禁止：

- 推翻布局
- 重做 IA
- 大面积渐变
- 赛博东方风
- 厚重传统风
- 大范围动画
- UI 各模块独立设计

---

# 十三、长期目标

Amber Compass 最终目标：

成为：

现代、可信、温润、专业的东方历法与命理工作台。

而不是：

“更高级的算命网站”。
