# Amber Compass Visual Token System

## 目标
建立 Amber Compass 的统一视觉基准，让 Logo、日历模块、八字模块、右侧操作区、空状态保持同一套视觉语言。

## 视觉关键词
- 现代
- 克制
- 温润
- 可信
- 专业
- 东方秩序感

## 核心原则
- Logo 是视觉母体
- UI 是 Logo 气质的延展
- 不做传统玄学网站风
- 不做赛博东方风
- 不推翻现有布局
- 不靠大面积装饰建立东方感

## 色彩 Token
待定义：
- 背景色
- 主卡片色
- 次级卡片色
- 边框色
- 主文字色
- 次文字色
- 琥珀强调色
- 选中态
- Hover 态

## 字体 Token
待定义：
- 品牌字体
- 模块标题
- 数据文字
- 辅助说明文字

## 圆角 Token
待定义：
- 小圆角
- 卡片圆角
- 面板圆角

## 边框与分隔线
待定义：
- 主边框
- 弱分隔线
- 激活边框

## 空状态语言
待定义：
- 轻点阵
- 浅圆环
- 低对比线条
- 短提示文案

## 交互状态
待定义：
- Hover
- Selected
- Focus
- Disabled
