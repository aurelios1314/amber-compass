# Task 014：Amber Compass 视觉基准 Canvas

## 背景

Amber Compass 当前已经完成：

- 基础布局稳定
- 八字验证体系初步稳定
- UI 产品层级整理
- 右侧操作区主次优化

同时：

Amber Compass Logo 已基本确定。

下一阶段目标：

不是继续局部修 UI，
而是建立统一视觉基准。

---

## 当前阶段

Phase 2：产品体验统一 / 视觉语言统一

---

## 任务目标

基于当前 Amber Compass Logo：

建立整体产品视觉基准。

重点统一：

- Logo 与产品界面关系
- 色彩体系
- 字体气质
- 空状态
- 卡片语言
- 日历 / 八字模块统一感
- 专业感
- 现代东方气质

---

## 视觉方向

整体方向：

现代
极简
克制
可信
温润
专业

参考：

- OpenAI
- Claude
- Google

但保留：

东方人文气质。

---

## 当前 Logo

当前 Logo 已确定方向：

- 双圆环
- 河图洛书点阵
- 云纹
- 山水
- 暖金 / 琥珀色
- 轻东方感

要求：

UI 必须围绕当前 Logo 气质统一。

禁止：

- Logo 与 UI 气质割裂
- Logo 高级但 UI 像后台系统
- Logo 东方，但 UI 偏冰冷科技风

---

## 重点分析区域

### 一、顶部栏

重点确认：

- Logo 如何进入产品顶部栏
- 品牌名权重
- 导航气质
- 留白
- 顶栏高度

---

### 二、日历模块

重点确认：

- 当前暖色系统是否与 Logo 协调
- 卡片语言是否统一
- 日历格子是否过于“系统化”
- 是否缺少品牌气质

---

### 三、八字模块

重点确认：

- 空状态是否像专业工具
- 四柱矩阵是否足够高级
- 右侧操作区是否仍偏后台感
- 八字页面是否像“可信工作台”

---

### 四、整体视觉语言

重点确认：

- 字体层级
- 分隔线
- 边框
- 圆角
- 空状态
- 按钮
- Hover
- 当前留白

---

## 不要做什么

禁止：

- 推翻整体布局
- 重新设计 IA
- 大范围改结构
- 传统玄学网站风
- 厚重拟物风
- 复杂动效
- 赛博东方风
- 大面积渐变科技风

---

## 输出要求

输出：

# Amber Compass Visual Baseline Review

包含：

## 当前视觉整体评价

## Logo 与 UI 是否统一

## 当前最大视觉问题

## 哪些区域最需要优化

## 日历模块问题

## 八字模块问题

## 空状态问题

## 字体与留白问题

## 建议优先优化顺序

## 哪些地方不要动

---

## 验收标准

任务完成后：

- Amber Compass 有统一视觉方向
- UI 与 Logo 气质统一
- 能明确下一阶段视觉优化路线
- 不进入无意义细节争论
- 不直接修改代码

---

## 执行要求

1. 先输出分析计划
2. 不修改代码
3. 不输出实现代码
4. 先做视觉基准评审
5. 等待确认后继续
